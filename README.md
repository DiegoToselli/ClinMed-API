# REST API - Clinica Médica

Simple REST API to manage doctors of a health clinic.

## Features 🚀

✔️ CRUD Operations\
✔️ HTTP Methods (GET | POST | DELETE | PUT)\
✔️ REST Controllers\
✔️ Authorization & Authentication\
✔️ MySQL database connection.\

## Used Dependencies

📘 Lombok
📘 Spring Data JPA
📘 Spring Validation
📘 Spring Security
📘 Auth0 - JWT
📘 Flyway DB
📘 MySQL Connector

## Made with 🛠️

📌 Java JDK 11
📌 Spring Boot
📌 MySQL Database
📌 Maven
📌 IntelliJ IDEA
